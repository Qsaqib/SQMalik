1. Change family type to another family type
   - This script will change one family type to another family type in the same category.
   - for example: If you have multiple walls of "Generic 200" and want all of them to replace 
   - by " Generic 150". You can simply do that using this script.

2. Create New Material
   - You can create new materials if you have data in the excel.

3. Checking Duplicates in materials
   - It checks if there are duplicate in materials by looking at their names and 
     various graphical properties
